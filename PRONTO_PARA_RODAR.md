## 🚀 COMECE AGORA (3 PASSOS)

### PASSO 1: Preencha o .env.local (5 minutos)

O arquivo `.env.local` já está criado na pasta do projeto. Você só precisa preencher com suas chaves:

**Abra:** `/tmp/sai-do-vermelho/.env.local`

**E altere estes valores:**

```bash
# Supabase - De https://supabase.com/dashboard
NEXT_PUBLIC_SUPABASE_URL=https://seu-id-aqui.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGci... (sua ANON_KEY)
SUPABASE_SERVICE_KEY=eyJhbGci... (sua SERVICE_KEY)

# Stripe - De https://dashboard.stripe.com/apikeys
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_51...
STRIPE_SECRET_KEY=sk_test_51...

# Anthropic - De https://console.anthropic.com/keys
ANTHROPIC_API_KEY=sk-ant-v0-...

# Deixar igual
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### PASSO 2: Rode o dev server

```bash
cd /tmp/sai-do-vermelho
npm run dev
```

Você vai ver:
```
✓ Ready in XXXms
Local: http://localhost:3000
```

### PASSO 3: Abra no navegador

```
http://localhost:3000
```
